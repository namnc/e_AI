- approval_phishing, l2_anonymity_set, l2_bridge_linkage, governance_proposal (close)

**FPR too high — analyzer over-flags (calibration needed):**
- cross_protocol_risk (29.2%) — ground-truth function counts mild-risk patterns as fully-risky
