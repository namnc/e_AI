# Domain: DeFi Query Sanitization (v1)

**Type:** Text query analysis
**CROPS property:** P (Privacy)
**Status:** Audited
**Schema:** `core/domain_profile.py` (DomainProfile)

## What this domain does

Sanitizes DeFi queries before they're sent to a cloud LLM. Detects and removes sensitive information (amounts, addresses, positions, strategies) so the query can be answered without leaking user data.

## Improving this domain

1. Add labeled sensitive spans to `data/`
2. Run validation: `python -m meta.validation_engine domains/defi/profile.json`
3. Compare with generated variants: `python compare_profiles.py`
4. See `docs/improving_a_domain.md`
